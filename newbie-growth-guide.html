新人联盟成长指南-全方位解读助您在台服冒险岛新枫之谷快速升级。

Tikas

教程

发布于：2020年3月15日

次浏览

对于台服冒险岛新枫之谷的联盟实力，本文从两方面去解读新人如何快速成长，一是花钱，另一个是不花钱！

一、职业前置解读

在台服冒险岛新枫之谷里，有着许多来自不同阵容的职业，他们有的可以直接从老家出来锻炼成长，有些却一定要完成老家的剧情任务，才能出来外面探险。

而更多的几乎是在 10级，就能在头顶提示一个灯泡任务：“〔教学〕来了解回家卷轴吧”，这是来自 “枫之谷GM” 给予勇士的 “回城卷轴”。

只要能在 10级完成这个任务，拿到 10张 回城卷轴，就不归纳在 “一定要完成老家的剧情任务” 里了。

在这里，小T同学已经为大家罗列出来，那些职业是在10级以后，还是要一定做完剧情任务，或者达到一定条件才能出来探险的职业：

  • 来自 “超新星” 的 卡蒂娜（国服：魔链影士）
  • 来自 “雷普族” 的 伊利恩（国服：圣晶使徒）
  • 来自 “雷普族” 的 亚克（国服：影魂异人）
  • 凯内西斯（国服：超能力者），出来魔法密林就可以了
  • 来自 “英雄团” 的 隐月

以上需要做完老家的前置任务，才能出来探险哦！

神之子需要完成第二章剧情之后，才能出来探险取得经验！

另外，还有许多，不用做完，直接在 10级完成人物头顶上灯泡任务，领取 “枫之谷GM” 给予勇士的 “回城卷轴”，就可以出来探险啦！

常见的有：

  • 剑豪
  • 阴阳师
  • 虎影
  • 开拓者
  • 阿戴尔
  • 等等…

有些职业不需要全部做完，比如：

  • 来自 “英雄团” 的 幻影，拿到经验药瓶喝完就可以离开了。
  • 来自 “达恩维尔” 的 幻兽师（国服：林之灵），只需要解锁四个形态就可以离开新手村了。

更多后续补充吧。。。

新角色，建议都保存一遍发型和脸型

  • 打开装备栏
  • 在装备栏里 点击 “SALON”
  • 分别在发型和脸型空白地方，点一下 “储存”

二、Link练号顺序建议

新号由于没有联盟属性支撑，前期，建议先练以下三个号到 120级（或者 140级），增加一些属性和伤害

如果您的 Link 装攻击高，可以直接想练什么就练什么

  • 第一个职业： “末日反抗军” 阵容的 “恶魔复仇者”
    2 级 link 传授是：伤害 +10%
  • 第二个职业： “晓之阵” 阵容的 “剑豪”
    2 级 link 传授是：全属性 + 25，攻击力 + 15
  • 第三个职业： “晓之阵” 阵容的 “阴阳师”
    2 级 link 传授是：伤害 +10%

以上三个职业，可以为后期 link 传授到 +20% 伤害，+25 点全属，+15 点攻击力，可以说非常不错了。

由于再多练加成属性的 link 传授会托慢进度，接下来，可以练三个经验加成的，进一步提高后续的成长速度。

  • 第一个经验加成： “英雄图” 阵容的 “狂狼勇士（国服：战神）”
    2 级 link 传授是：连击宝珠经验 +650%
  • 第二个经验加成： “英雄图” 阵容的 “龙魔导士（国服：龙神）”
    2 级 link 传授是：符文持续时间 +50%
  • 第三个经验加成： “英雄图” 阵容的 “精灵游侠（国服：双弩精灵）”
    2 级 link 传授是：经验值获得量 +15%

有了三个基础属性加成，和三个经验加成，后续再练其他职业就轻松快捷多了

更多 link 传授可查阅：
点我查阅台服冒险岛新枫之谷 link 图

如果前期新号没 link 装，建议先练到 70级，获取 1级 link传授

三、花钱成长

钱要怎么花，这是一门学问，新人刚入谷，如果是花钱流，联盟的成长，又可以分为两个：一是找别人代练，二是自己练。

千万不要去购买别人账号！由于账号都是绑定个人信息，别人随时可以找回！


  • 1.花钱找人代练
  • 2.花钱自己练
  • 3.睿智葫芦升级方法

优点是可以省时间，缺点是没能亲自去体验升级过程

价格一般是：

  • 6000联盟 2500元左右
    （不含商城双的大概 100元钱，以及最好为四个大职业群各配一个宠物，也就是四个宠物大概 50元）
  • 8000联盟就要贵些，大概 8000元左右
    （同样也是不含商城双，一个号大概两张商城双就够了）

如果是直接开练 6000联盟 以上的，不用考虑蓝双（非商城的双倍），代练会帮签到 Happy Day 换取蓝双。

在这里说声，如果不是特别急，不建议杀宠物换取蓝双，毕竟这代价太大了！

自己买轮回碑石和燃烧之戒给工具号（另外注册一个号，一般拿林之灵或者主教来当工具人）

再收购 Link 装和联盟装，自己慢慢练。

这个好处是，轮回碑石基本上是大号必备品，属于必买品，燃烧也不错，这两个加起来可能就要用掉 2000-5000元（价格因市场因素决定）。

Link 装上的卷不同，价也不同，一般是按一套多少攻来算钱，法装便宜些，物理和法师各一套，总价格大概在 1500-2000之间（价格因市场因素决定）

联盟装就要分不同职业了，五大职业群，都要备一套，成本高，可以考虑租凭。

这里成本投入近万元，好处是体验了各职业的特色和练级，并且，投入的后期可以转手出售，算是 “零成本”，只花了自己的时间。

如果你有 “睿智葫芦” 道具，你可以选择一个 200等级以上的角色做为导师，吸满同等级的 10W 个怪物数量（不一定要自己打），就可以给小号使用啦！

注：导师角色等级一定要大于 200等级！

小号使用时，只需要升级到 50等级，就可以直接从 50级 升到 200级 啦！

不管是自己练还是请别人代练，下面的练级方法多看看，也是有好处的

# 15-40 在小青蛇渡过

前期可以在 “堕落城市（CMS：弃废都市）” 的 “沼泽地” 练小青蛇渡过到 40级

如果你的装备不够给力，可以在魔法密林打独眼怪到 25级，再回来这里练级

# 40-60 在土龙渡过

40-60级可以在 “奇幻村（CMS：林中之城）” 的 “沼泽：宁静的沼泽” 练土龙过渡到 60级

如果你的 Link 装不足以秒怪的情况下，建议在这里升级到 62 级，再去三转，三转之后，去沙漠升级到 73级，之后直接去下面玛迦提亚城的隐藏地图：闲人勿入

# 60-86 在沙漠渡过

三转之后，如果您没有趁手的武器，可以先到 “纳希沙漠” 的 “纳希绿洲城” 里找一个 “渣伊德” 的武器商人，可以购买到很多职业的 60级武器。

之后可以在沙漠的 “日落之路” 练级。

其中：“沉睡沙漠” 和 “萨赫勒地区2” 都是地形平坦，他们之间的区别是：

  • 沉睡沙漠地图小，正常怪物数量：13个
  • 萨赫勒地区2地图一般，正常怪物数量：15个

# 86-112 在天空之巢Ⅱ渡过

“神木村” 的 “天空之巢Ⅱ” 又简称 “鸟2”， 是最轻松的星图练级地方，对装备的星力要求不高，只需要：五颗星。并且能得到很多的经验值，非常不错！

小技巧：你可以升级到 111等级，经验在 85%-95% 之间，这样子过去小火狗一到 112等级，就可以爆一下，其他等级的也适用.

# 112-140 在小火狗渡过

在 “冰原雪域” 的 “试炼的洞穴1”里，这是一个很好的练级地方，如果职业很容易掉血，可以在中间泡熔浆来规避怪物的攻击。

# 140-160 在星光之塔渡过

坐落在 “维多利亚岛” 的 “星光之塔”，每年都接受到许多冒险家们的青睐。最受欢迎的是 “2楼咖啡厅” 以及 “5楼化妆品店”。

在这里冒险勇士们可以很快成长到 160-180级，建议实力强的职业，160级就可以离开星光之塔了，如果实力弱的职业，可以在这里享受多一些。

# 160-180 在外星人渡过

地球防御本部外面的 UFO飞船，拥有着非常多的外星人。其中，UFO 内部的走廊 H01 更是时时刻刻都能看到许多冒险家们，这里的外星人攻击非常高，建议要有良好的装备条件再来这里冒险。

如果你的职业并没有什么冷却快的全屏群攻技能，那么，这里升级到 200 等级是最好的选择。

# 180-200 在黄昏的勇士之村渡过

“时间神殿” 有三个门，其中一个叫 “未来之门” 里有个 “黄昏的勇士之村”，里面有两个地方是非常好的练级地方。

  • 被遗弃的挖掘地区2
    练级神图，怪物集中，非常多冒险家在此出入
  • 荒郊野外
    一个被大家快遗忘的地方，如果你在挖2没有图，不如来这里试试看

只需要到达 199级，经验值在 47% 就可以停下来！

去做左边灯泡里 190级的任务（来自新村庄的信件），一直对话般做下去，不用三分钟，就可以到达 200级

四、不花钱成长

虽然不推荐这么肝，如果可以，您可以选择租 link 套和租轮回来练，这样花费小。

首先，要知道，这里不是国服，所以，我们要走一条强化之路，自强不息！

新号前期需要打野捡装备，捡咒文的痕迹，以方便自己快速成长，并且有一定资金和咒文的痕迹，可以更方便下一个号的速成。

这里以练 6 个小号到 70 级，取得 1级 link 传授

从第 7 个号开始，为自己所选的大号（本尊主号），这样，当大号到达 80 级时，可以开启联盟！

  • 一、首个角色前期积累
  • 二、前期职业快速到 70级
  • 三、主号快速到达 200级

  • 10级出来去 “弓箭手村” 右边的 “芽孢山丘”，接绿菇菇悬赏标示牌，完成任务取得：耳环装备

  • 10-25 在 “魔法森林” 的 “北部森林：绿树的藤条” 打风独眼兽。
    记得要捡的东西：药水、装备、咒文的痕迹，前面两个可以回城拿去卖钱，这样可以快速积累资金

  • 需要准备的装备：
    1、0-18级的 帽子，购买价：3800，7次强化
    2、0-18级的 上衣，购买价：8500，7次强化
    3、0-18级的 裤子，购买价：8000，7次强化
    4、0-18级的 鞋子，购买价：4000，5次强化
    5、0-18级的 手套，购买价：4000，5次强化
    6、0-18级的 耳环，任务取得，5次强化
    7、自带的 10级 武器，7次强化

如果打野能捡取到，就可以省下，不用去城里购买（没有自己的防具装备，可以去维多利亚港购买通用的）
每次强化需要1个咒文的痕迹，强化5星需要大概 8000游戏币

如果强化 20级以上的装备，每次强化，将需要 2个咒文痕迹！

  • 需要准备的游戏币和咒文的痕迹
    游戏币最好能有：12万，以防装备需要去购买，或者咒文痕迹不够时，好去拍卖购买
    咒文的痕迹最好有 43个，可以通过击败精英怪快速获取。后续不够，可以通过拍卖购买，一个大概 3000游戏币


达到 20级，优先给武器强化，并上满 5星。

强化只需要 100% 咒文的痕迹，手套可能只有 70% ，就强化 70%概率的。

之后有 4万左右游戏币，就去 菇菇神社 购买 5分钟的 攻击药水。



角色没有 100级，前往菇菇神社方法

首先回到城里，在左上角的小地图下面点击快速移动到售票处，和移动小帮手对话就能去菇菇神社啦！

在菇菇神社里找到 “章鱼烧厨师：健二” 购买对应的 5分钟 攻击药水

  • 物理：加量章鱼烧，每个 4200
  • 魔法：双份日式炒面，每个 3800

之后，可以通过 “水晶” 回到 “弓箭手村”




有了攻击药水，接下来，就尽量把帽子、上衣、裤子、鞋子、手套、耳环等部位都强化 100% 咒文的痕迹并且上满5星

进入成长阶段！

后续练其他小号时，可以把第一个号的物资通过仓库转移
以方便小号前期提前做装备，进入成长阶段

前期联盟小号，主要以快速到达 70级为主

故，这里先暂时介绍 70级内的练级方法

# 10-15级 在魔法森林-和天空临近的地方 刷 蝴蝶精

由于出来新手村需要去 “弓箭手村” 右边的 “芽孢山丘” 做任务拿耳环，拿到之后，可以直接过来这里刷蝴蝶精快速到 15级

# 15-30级 在魔法森林 的北部森林 里刷 风独眼兽

这里开始，尽量有装备、药水、咒文的痕迹，就捡，装备满了就回城卖，等级到 20级，就强化武器，之后可以慢慢补全其他部位。

30级二转之后，一定要有装备，有攻击药，没有请去菇菇神社购买。

# 30-40级 在堕落城市 的沼地：沼泽地 里刷 小青蛇

由于攻击并不是很充足，请一定要保持 5分钟的攻击BUFF 不会断，以加快清怪速度

记得有点数要用，加攻的被动先点满。

# 40-69级 完成三个主题副本：妖精学院、黄金海岸、里艾娜海峡

从 40级 开始，快速完成这三个副本，出来可以达到 69级

每完成一个副本，会得到一些装备奖励，缺少的装备或者更好，可以强化替换

# 69-70级 去60级指导的 “遗迹发掘地”

在这里完成三个 “遗迹发掘地布告栏” 的任务，就可以很快到达 70 级了。

键盘里没有 “指导” ？请按下面操作

按 Esc 键，在弹出的菜单里选择 “安装快捷键” ，把 “指导” 拿到上面空白的按键里，保存退出。

这样就能快速按两次那个按键，打开指导界面了。
