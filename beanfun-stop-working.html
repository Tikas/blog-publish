放心！

本文从：如何下载、如何使用、以及一些常见问题解决等，全方位解答这款软件，不用担心在一些不安全的地方下载，也不用担心各种出错报错的问题没法解决。

二、内置实用的小工具

登录缤放之后，在主界面的右边，可以找到一个 “工具” 按钮，里面有一些很实用的小工具哦！

最实用的装备卷轴星力计算器

使用方法是：选择对应的装备部位，对应的等级，在 A 和 B 里填写装备自身的属性，就可以啦。

其中：
A、装备当前星力强化等级（就是多少星的装备啦）
B、是装备原本白色的属性值，这个一定要填写哦，不然像武器类他的攻击加成是按装备自身多少攻击来算的。
C、这是星火属性值，写上去，如果最终上卷正确，就能在左边的总值和装备对得上，不填写这个就会少星火属性。

也许最终加上卷的值，装备的属性会比计算器的多一些攻击，这是因为天然白板装时，就有浮动的属性加成。

4.2、没有添加账号，启动游戏会提示：未选择启动游戏的账号

如果你的账号是刚刚注册的，登录缤放之后，是无法获取密码的，你还需要进行下列操作：

打开查看：在缤放里增加账号

1.在缤放主界面右下角，点击 “新增账号”

2.在 “使用者名称” 里输入注册账号时填写的昵称（其实可以随便填的，但是发生过 BUG，这个 BUG 应该不会再出现了。）

4.新版本缤放默认是隐藏了账号，你可以在账户名称里双击复制你的账号。（如果选中自动输入，获取密码后，缤放会自动帮你输入账号和密码）

就可以解决这个问题啦！

如果 NGS 一直提示，不会消失，那么请重新下载安装一次游戏

NGS 一直提示，不会消失，多是游戏文档出现破坏，导致的。

出现这种情况，就需要卸载游戏，重新安装一次了。

如果重新安装之后还是这样，那么，多数情况下，您下载的游戏文档是不完整的，请重新下载一次游戏文档，以方便进行重新安装游戏。

这是由于缤放登录器默认是阻止自动更新的！

5.2、 提示:无法加载进行反序列化所需的类型 beanfun records

从 3.0.1 版本开始，Pungin 作者重新更新了本地记录文档，旧版解析新版本的本地记录文档就会提示 “无法进行反序列化”

打开查看：解决无法加载进行反序列化所需的类型

由于作者在 2020年4月7号进行更新了 3.0.1 版本，如果有同学进行更新了，并且出现下面两个提示：


那么，您的缤放，应该是更新完之后，没能替换原来的旧版本，或者是桌面的快捷方式里的版本未能进行替换

导致：

运行旧版本（3.0.0版本之前的）都将会提示 “Beanfun Records！” 错误。

解决办法是前往作者的 Github 下载最新版本的缤放，就可以解决这个问题。

第三方缤放 Github 下载页面

https://github.com/pungin/Beanfun/releases

不建议删除账号记录文件以启动旧版本为目的的操作

删除缤放账号记录文件方法:

按 “Win + R” 键打开运行，在运行里输入：%appdata%

在 “AppData\Roaming” 里删除 “Beanfun” 目录就可以。

如非必要，请不要删除此文档，不然，您所记录的所有账号密码将清空！

5.5、非繁体语系系统启动游戏失败，提示：C00700C1

有三种原因导致这情况出现：

  • 1、使用了自动更新，导致游戏文件被损坏，请重新安装一次游戏或者找朋友另外复制一个游戏主程序过来试试（建议大家别用自动更新）

  • 2、更改了语言区域导致区域异常，也会提示这个，解决办法就是简体语言和繁体语言换一换来尝试看看了

  • 3、被安全软件拦截了 LE 模块，比如使用了 3** 相关安全软件。建议到 msdn itellyou 里下载镜像重装系统！

5.6、缤放启动游戏提示错误码：C0000135

如果你的缤放启动后提示：C0000135，这可能是缤放的 LE 模式被相关安全软件拦截，或者你的系统本身之前有修改过其他语言区域。

解决办是修改回简体语言环境，或者改为繁体语言环境

如果你想改为繁体语言环境，可以参考本文的前面：查阅:三、两种启动方式（一、使用网页进入游戏）

5.7、缤放无法登录账号（3.0 之前的旧缤放版本）

如果你的缤放登录器是 2.9.9 版本之前的，那么提示无法登录账号，就是没有安装橘子官方的 beanfun 平台程序导致的

如果是 3.0 版本之后的，那么你可以看看是否网络问题导致的，有时 网络故障 就无法登录了，请换加速器看看。

Zip 压缩文档里面会有这两个 dll 文件，把这两个 dll 文件拷贝到缤放登录器的同一目录，就可以解决此问题

六、网络相关导致的常见问题

这里出现的问题，多是由于加速器或者其他网络因素导致的

当然

也有特别例子，如账号服务不存在等情况

6.1、缤放无法获取密码

有多种原因会导致这种情况出现：

  • 1、未安装 beanfun 程序，或未连接上程序：
    没有安装，可以看上一个问题解决方法，如果已经安装未能连接上，可能需要重新安装一次程序看看，如果还不可以，或许是被某些安全软件拦截了。
  • 2、未启用台服冒险岛加速器导致，或网络问题：
    有些区域，可能需要开启加速器之后，才能正常获取密码，往往先开加速器，还能有效避免锁号问题。
  • 3、2020年5月底因大量工作室涌入导致暂时关闭新号密码获取：
    这是短期的，等这情况过后，就会恢复。

6.2、缤放无法打开会员中心、充值页面等

这也是加速器不支持导致的，建议更换一个加速器，或者不开加速器试试看。

6.3、登入失败，未找到 Eventvalidation 讯息，请检查网络连接

如果出现这情况，请留意一下账号安全情况！

登入状态除非很久，或者更换过节点等操作，正常情况下是：不会执行退出账号行为的！

那么，您刚登录账号不久，再获取密码就提示这个，应该是有人登录过你的账号了！

具体登录记录可以从：会员中心-查询记录-大事纪-beanfun!主账号 查询账号登录情况。

6.4、缤放提示：获取密码失败，服务账号不存在

之前就有多个萌新深夜去注册账号，无论等待多久都提示：“服务账号不存在”，但是查询账号信息都显示状态正常。

经过多位小伙伴提供的注册时间可以得出：深夜后注册的账号等于空号！这时候您就需要通过 “客服中心” 联系客服，帮忙解决此问题

请大家别在深夜去注册账号。

另外，每次维护时注册的账号也会出现这情况，维护时间多是在星期三，这是由于数据同步不到游戏数据库，这情况就需要等待一天了。

要是等了一天多，还是不行，就需要:联系客服进行账号异常处理

著名服务器卡爆事件也会导致此类情况发生

2020年5月底大量工作室进台服，导致服务器卡爆，官方直接把月底几天申请的新号暂停密码取得服务，并且加强在线 GM 在线排查。这时候申请的账号就会获取不到密码。

后续会不会再次出现这种情况也很难说，好在这种特殊情况只维持几天。

6.5、获取密码失败，请尝试重新登入。索引和长度必须引用该字符串内的位置。参数名：length

这情况多是 beanfun账号注册不久发生的，因为港服的号要同步到台服数据库里，这需要一定的时间，有时遇到数据库维护，就要等多几天了。

太难了。。。

七、锁号误封或者永久封号的账号无法进入游戏

如果账号显示灰色，要么是锁号了，要么是封号了。

请查看原因，可以点 “会员中心” - “查询记录” - “大事件” 里查看。

之前大事件不能查询当天的，最早是昨天的。现在不知道是否可以查询当天的了。

事件显示：帳號鎖定(終止遊戲契約(永久鎖定);0)

出现这情况，会有两种情况，一种是使用非法程序导致永久封号，如：压键盘、使用鼠标连点、外挂程序、购买工作室游戏币，等等。

如果不是上面所说的情况，那么应该是误封了，请申请解锁。如 2020年3月份 有名的大面积永久误封斗燃猫号，只要申请，说明原因，就会得到解锁。
