一、游戏下载

如果您同我们一样都是使用加速器来玩新枫之谷的话，那么下面这点一定要记住：

使用加速器体验游戏的小伙伴请手动更新

当游戏出现更新包时（或者缤放提示了自动更新可能破坏程序，已被阻止），请您一定自己去手动下载更新包进行更新：

点我前往： 官网更新包发布页面

如果官网更新包发布页面没有最新版本的手动更新包，可以从官网的最新消息里找到。

因为使用加速器在线更新容易丢包，导致更新失败，从而破坏游戏文件！

如果已经在线更新中，可以结束更新进程停止在线更新，使用手动更新包本地更新一下就可以。

这方法同样适用游戏文件被破坏时进行补救！

如果游戏的目录路径带有中文，您将无法正常启动游戏！

错误的安装目录例子：C:\新枫之谷\MapleStory.exe

三、注册 beanfun 平台账号

由于港服和台服已经合并。现在的新枫之谷，是同时支持港号和台号在线游戏的。由于台号已经关闭相关验证，我们主要以注册港号去玩新枫之谷。

所以，我们先弄清什么是台号，什么是港号。

台号：也就是台服冒险岛新枫之谷官网里注册的账号，他需要验证台湾号码等相关信息，注册难度大。

港号：香港专区已经没有新枫之谷的官网了，现在港号其实就是香港 beanfun 平台的账号，他一样能玩新枫之谷。

重要提示！

A）如果您是在深夜注册的，可能会出现账号一切正常，获取密码的时候，一直提示账号服务不存在！请别在深夜注册账号！

B）如果您是在维护期间注册账号，也可能会出现提示账号服务不存在！请别在星期三的 00：00-15：00 时段进行账号注册！

C）由于大量工作室涌入，在2020年5月底注册的新号，将暂时无法取得密码！现在已经恢复，后续难免还会出现此情况。

如果您真的提示了：“服务账号不存在”。请查阅解锁教程进行联系客服，问题请选择 beanfun 账号相关，并如实描述。

这里提供两种注册方法，其实都是通过同一个网页注册的账号，只是使用第三方缤放在注册上体验感会好很多。

为了自己的账号安全，不建议使用 QQ 号码的邮箱做账号！例如：12345@QQ.com —— 重要提示请记笔记！

3.1、使用网页进行账号注册

由于港号使用了较旧的技术，只支持 IE 浏览器。这一步的所有操作都将在 IE 浏览器里进行，请确保您所使用的是 IE 浏览器！

打开查看：使用网页进行账号注册

第 1/3 步，去掉 IE 浏览器的保护模式

打开 IE 的 Internet 选项，在安全里去掉 “启用保护模式”，保存并关掉 IE ，重新打开 IE 就生效了。

如下图所示：


第 2/3 步，打开账号注册页面

香港橘子 beanfun 账号注册页面

http://hk.beanfun.com/locales/HK/contents/beanfun_block/help/plugin_install.html

第 3/3 步，填写相关信息

点加入会员，进行账号注册，整个信息建议要真实，可以用 QQ 邮箱注册。(如果你的邮箱无法收到邮件，请尝试更换别的邮箱注册)

如果提示网页未显示完整内容，请选择显示所有内容。

以下请记好笔记

上图有三点是非常重要的，特别是“保安问题”，在修改密码时，是非常重要的验证！

  • A 为港号的账号，这是橘子平台的账号，不是游戏账号，我们是先登录游戏平台再启动游戏的。
  • B 为昵称，算是游戏账号的别名，随意，简单些，可中文，注册完成后添加账号时，请填写这个昵称！（之前出现过未填写此昵称无法正常进入游戏 BUG，建议请填写这个昵称）
  • C 为重中之重，这答案关系到你能不能快速方便在忘记密码时修改密码
  • 出生日期不能乱选择，必需要和自己证件号码相对应，不然注册的账号无法进入游戏

为了后续方便解锁申请，请真实填写相关个人信息
以上 A 和 C 非常重要！
以上 B 为昵称，并非账号！

注册完会提示你前往自己的邮箱验证，验证完之后，就算是初步成功啦！

3.2、下载第三方缤放登录器，并进行账号注册

第三方缤放登录器是一个非常好用的工具。然而，由于并非官方发布的，也有不法人士进行对他修改捆绑了盗号程序，让开发者非常难过。

如果您同我一样，非常喜欢使用登录器玩游戏（体验感非常好），建议您一定要去作者官方的 Github 下载！

由于第三方缤放软件并不像猪头那样取一个别名，第三方缤放不仅图标同官方一样，连名字也是，这会让超多的人会误认为这是官方的不说，还会让一堆的萌新分不出那个是真，那个是假（虽然都是真的）

在这里特别声明一下：

官方的缤放是一个平台的服务程序，名字是：beanfun!

第三方缤放是一个登录程序，名字是：beanfun

他们唯一区别是：是否带一个 “！”

很多人说本文没有第三方缤放登录器下载地址，请点开下面：使用第三方缤放登录器进行账号注册，查看！ —— 重要提示请记笔记！

打开查看：使用第三方缤放登录器进行账号注册

第 1/4 步，前往作者 Github 页面下载

第三方缤放 Github 下载页面

https://github.com/pungin/Beanfun/releases

第 2/4 步，下载缤放登录器

在这个页面的内容最上方，一般就是最新版本，直接点击 “點擊下載「繽放」” 就可以下载了。

如果无法下载，可以尝试点击 Assets 里的：Source code（zip）

五、启动游戏的 3 种方法

beanfun 账号都认证了，是不是很想尝试进入游戏呢？

别急，你还需要知晓三种启动游戏的方法之一，才能正常体验游戏。

  • 一、使用网页进入游戏
  • 二、使用第三方缤放登录器进入游戏
  • 三：使用 LE 启动游戏

要想使用网页启动游戏，你需要把电脑的区域设置为繁体台湾。

打开：控制面板，在控制面板里找到区域。如果你的是 “类别” 显示，你应该先找：时钟和区域，再在里面找到 “区域”

在区域设置的 “管理” 非 unicode 程序的语言里，点击 “更改系统区域设置”，选择设置如下图：

设置繁体区域之后，建议不要使用缤放来启动游戏啦，不然容易报：C00700C1 错误。

这样就可以直接启动台服冒险岛新枫之谷啦。

使用 “第三方缤放” 可以很好解决 “This client can be executed only in Traditonal Chinese code-page” 这类问题，因为 “缤放” 已经引用了 Locale Emulator 模块，简体系统，一样玩新枫之谷！
